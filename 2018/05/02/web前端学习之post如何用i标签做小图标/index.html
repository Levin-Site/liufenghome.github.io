post如何用i标签做小图标

发表于 2018-05-02 | 分类于 学习笔记

1.下载矢量图代码

地址:阿里巴巴:http://www.iconfont.cn/

1.登录后找到你喜欢的图片,加入购物车
2.点击自己的购物车
3.选择下载代码

2.解压，打开unicode

3.使用类似如下代码

<i class="iconfont">&#x33;</i>

4.clss类必须包含 iconfont ,其中的内容跟实例 unicode编码相同即可
